# Architectural decision records

This directory holds one record per decision, named `<number>-<slug>.md` (for example `001-event-bus-choice.md`). A decision record is created only when the decision is all three of: hard to reverse, surprising without context, and the result of a real trade-off. Keep each to a few sentences.

## Decision record format

```
# <number>: <short title>

Context: the situation and the forces at play, in one or two sentences.
Decision: what was chosen.
Why: the trade-off that made this the choice, in one or two sentences.
```
