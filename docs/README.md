# Framework documentation

Documentation about the agentic SDLC framework itself. None of this is copied into a target repository; it explains how the framework is designed and how to drive it.

- `methodology.md`: the conceptual model of the framework, the design thesis, the phase model, the implementation loop, and the review panel.
- `getting-started.md`: how to bootstrap the framework into a repository, run setup, and drive the first pipeline end to end.
- `phase-guides/`: one guide per phase, covering what each phase reads, what it produces, and how to run it.
